# PROYECTO FINAL JAVA BACKEND INTERMEDIO - BEDU

## API Veterinaria

El proyecto se centra en una API de una veterinaria desarrollada en Java
con Spring y se busca facilitar la gestión de consultas y tratamientos de mascotas. 

## Tecnologías
- Java 17
- Spring 3.2.0
  - Spring Web
  - Spring Data JPA
  - Lombok
  - MySQL Driver
- MapStruct 1.5.5
